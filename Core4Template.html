Variables Something that stores designated values Drawers in a dresser contain different things like one designated for shoes or t-shirts User input like defining a value for a question Answer blocks or operator blocks (like division and addition)
Loops A repeated sequence of instructions a decided amount of times Jump-roping or a daily commute/routine using "while" or "for" to repeat a set of instructions indented inside the loop using a "Repeat __ times" block or "Forever" block to show how many times instructions within the block should repeat
Conditionals A Conditional is a statement that gives dependency and a direction to something, showing what would happen if or if not. If it is raining, then I wear my rainjacket or use my umbrella, else I will get wet. You can use the "if" command to show what happens with user input and anticipate results and develop commands for those results. if __ = yes: print("Yes, please!") if __ = no: print("No, thank you!) else: print("Invalid Response, sorry!") using the if then block or if then else block. If then is more general, and if then else is less limited
Functions A function can be a specific custom set of operations defined to simplify code or an action. A verb Defining a draw shape function to perform the operations in less code Defining an action with purple function block
